# Everything Companion Service

This optional Windows-only companion service exposes a small localhost API for the Search this PC with Everything screen in Organize Your PC.

The published Windows x64 package includes the official Everything SDK runtime (`Everything64.dll`) and its licence notice. The Everything application itself must still be installed and running separately.

Installation is successful only after the verification script confirms that the Companion can reach the running Everything database.

For development details, see the source files and the release workflow.
