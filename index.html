Organize Your PC

Personal Memory-Based File Archiver

Build a folder tree based on how you remember your files, then archive files or folders manually with full control.

This app is for personal use only. As a good practice, we recommend that you always keep a backup of your important files. This app creates copies and does not delete original files or folders.
Optional: Use Everything, a free companion tool, to find files and folders faster before archiving.

Choose what you want to do

Build New Folder Tree on this PC

Build a memory-based folder structure. The first level is fixed; all deeper layers are user-defined.

Before adding folders, choose one thinking type for the next layer. Chronological, thematic, functional, or role-based. The type guides the folder names.

Disclaimer

        
      


      

        
Organize Your PC is a personal memory-based file and folder organizing and archiving app.

        
It does not delete, upload, rename, modify, or automatically move files or folders.

        
Optional Google Analytics usage statistics are loaded only after the user allows them. File names, folder names, paths, and file contents are never included in the app's analytics events.

        
The app can archive a loaded file or chosen folder only after the user chooses a destination folder and gives browser permission.

How This App Works

        
      


      

        
The app has four main choices.

        
    
          
  1. Build New Folder Tree on this PC: create and manage your memory-based folder structure.
    2. 
          
  2. View Existing Folder Tree on this PC: choose a local folder and build the folder tree up to the depth you select.
    3. 
          
  3. Archive a File: load a file, select a destination from the folder tree, and archive it there with your permission.
    4. 
          
  4. Archive a Folder: choose a folder, select a destination from the folder tree, and archive a recursive copy there with your permission.
    5. 
        

        
Only one mode is active at a time.
