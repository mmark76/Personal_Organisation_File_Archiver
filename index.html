Organize Your PC

Personal Memory-Based File Advisor

Build a personal folder tree and get guidance on where to archive each new file. The app does not reorganize files on your PC. It does not move, delete, upload, or modify files.

Important Disclaimer

Organize Your PC is a file organisation advisor for your computer.

It does not reorganize your existing files on your PC. It does not move, delete, upload, rename, or modify any files.

The only optional action the app can perform on your PC is to create a new folder tree inside a destination folder that you choose.

How This App Works

Organize Your PC helps you organise your files by first helping you build a clear folder tree.

The app does only two things:

  1. It builds a folder tree based on the folder names and structure you define.
  2. It guides you on where to archive a file, using the file title and other available parameters.

Before the app can suggest where a file should be placed, the folder tree must first exist. Once the tree is ready, the app can compare a file, such as the built-in example file CV.pdf, with the available folders and suggest a useful destination.

The app does not interfere with your existing file organisation. Existing files can remain where they are, or they can be kept separately in an archive folder. From that point onward, you can start using the new folder tree for clearer future filing.

The purpose of Organize Your PC is not to automate your computer. Its main purpose is to help organise the way you think about your files.

The only optional automation is the creation of a new folder tree inside a destination folder chosen by the user.

The app does not move files.

Folder Tree

Fixed first level. Add unlimited layers below it.

Before adding folders, choose one thinking type for the next layer. Chronological, thematic, functional, or role-based. The type guides the names; it does not become a folder.
